Ruby gem to communicate with new Basecamp API.

###Documentation
Documentation is [here](http://rubydoc.info/github/birarda/logan/).

###Examples
```ruby
logan = Logan::Client.new("12345678", "username", "password", "LoganUserAgent (email@example.com)")
logan.projects
````
